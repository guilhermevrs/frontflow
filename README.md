### Projects incorporated
* [Harvest - Gulp boilerplate](https://github.com/ryanbenson/Harvest)

### Install
```
npm install
```

### Run
* To run the gulp generator
```
gulp
```
* Open the browser in http://localhost:8080

* To deploy your project
```
gulp deploy
```
